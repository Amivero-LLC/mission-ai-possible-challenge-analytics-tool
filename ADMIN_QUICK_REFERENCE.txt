HOW TO RUN ANALYSIS (3 EASY STEPS):
================================================================================

STEP 1: EXPORT CHATS FROM OPENWEBUI
   1. Go to OpenWebUI Admin Panel
   2. Click "Export All Chats"
   3. Save file (keeps timestamp in name automatically)

STEP 2: RUN ANALYSIS
   Option A: Double-click "RUN_ANALYSIS.bat" (Easiest!)
   Option B: Run command: python analyze_missions.py

STEP 3: VIEW DASHBOARD
   - Dashboard opens automatically in browser
   - Or manually open: mission_dashboard.html

         âœ… Check "Add Python to PATH" during install

Solution: Check folder permissions (need Read/Write)
